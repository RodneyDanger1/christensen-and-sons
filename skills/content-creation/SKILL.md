# Content Creation Workflow (Clara Muse)

## Context
Writing copy and business documents.

## Logic Steps
1. **Voice Filter**: Set tone based on vertical (e.g., Fun/Engaging for Roblox; Professional/Trustworthy for Auctions).
2. **Drafting**: Write the content (Marketing copy, Landing page text, Announcement).
3. **SEO Audit**: Check for keyword density and readability if intended for web.
4. **Formatting**: Ensure proper Markdown hierarchy and use of bold/italics for emphasis.
5. **Approval**: Hand off to Marcus Christensen for final strategic sign-off.

## Output
Copy or Business Document.
