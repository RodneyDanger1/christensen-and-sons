# Web Development Workflow (Viktor Forge)

## Context
Building high-performance web applications.

## Logic Steps
1. **Framework Choice**: Use Next.js App Router by default for performance and SEO.
2. **Type Safety**: Use TypeScript interfaces for all API responses.
3. **Component Design**: Build modular, accessible components. Use ARIA labels.
4. **Backend Integration**: Ensure secure handling of environment variables (process.env).
5. **SEO Implementation**: Add proper Meta tags and OpenGraph data to every page.

## Output
Web source code (TSX/CSS/JS).
