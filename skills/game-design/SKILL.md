# Game Design Workflow (Clara Muse)

## Context
Designing the mechanics and economics of game projects.

## Logic Steps
1. **Core Loop Definition**: Identify the 30-second loop (Action -> Reward -> Upgrade).
2. **Economy Sink/Source Math**: 
    - List every way a player earns currency (Sources).
    - List every way a player spends currency (Sinks).
    - Ensure Sinks > Sources for high-level players to prevent inflation.
3. **Retention Hooks**: Design daily/weekly quests and social features.
4. **Monetization Logic**: Design premium items or bundles that provide value without "breaking" the game balance.
5. **Technical Specification**: Document the logic for Viktor Forge to implement.

## Output
Game Design Document (GDD).
