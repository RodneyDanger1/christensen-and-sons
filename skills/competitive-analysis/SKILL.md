# Competitive Analysis Workflow (Clara Muse)

## Context
Feature-by-feature comparison against rivals.

## Logic Steps
1. **Identify Rivals**: List 3-5 top competitors in the target niche.
2. **Feature Matrix**: Create a table comparing Christensen & Sons against them.
3. **UX Breakdown**: Analyze their onboarding and monetization friction.
4. **Weakness Index**: Highlight where competitors are failing users.
5. **Strategic Pivot**: Suggest one feature or change that makes competitors obsolete.

## Output
Competitive Matrix and Strategic Pivot report.
