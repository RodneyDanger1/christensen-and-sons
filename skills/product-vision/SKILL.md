# Product Vision Workflow (Marcus Christensen)

## Context
Marcus determines the "Soul" and scope of a new product or feature.

## Model Grounding (Nemotron 128K)
- **Competitive Synthesis**: Synthesize departmental expertise from memory to define a 10-star experience.

## Logic Steps
1. **10-Star Version**: Describe the "ideal" version that exceeds user expectations.
2. **MoSCoW Prioritization**: Categorize features into Must, Should, Could, and Won't have.
3. **Strategic Filter**: Ensure alignment with Christensen & Sons' lean, self-hosted brand.
4. **Product Concept**: Produce a conceptual brief for Clara Muse.

## Output
A Product Concept document.
